Welcome to the Earthquake Finder!

This application allows you to track earthquakes based on the country and the number of days before today. The data displayed includes the country, place, magnitude, and time of the earthquake.

How to use the application:

  1. Enter the country code in the 'Country' field. For United States enter the states code instead results.
  2. Add the number of past days you want the data for in the 'Past days' field (for example, 7 for data from the past week).
  3. Click 'Search'.
